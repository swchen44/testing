# WiFi Bora Base Expert

WiFi Bora domain 基礎 Expert，提供協議、架構、build 和記憶體相關知識。

## 安裝

```bash
python connsys-jarvis/setup.py --init wifi-bora/experts/wifi-bora-base-expert/expert.json
```

## 包含的 Skills

- `wifi-bora-protocol-knowhow` — 802.11 協議知識
- `wifi-bora-arch-knowhow` — 韌體架構知識
- `wifi-bora-build-flow` — Build 流程 SOP
- `wifi-bora-memory-knowhow` — 記憶體管理基礎
- `wifi-bora-linkerscript-knowhow` — Linker script 知識
- `wifi-bora-symbolmap-knowhow` — Symbol map 解讀
